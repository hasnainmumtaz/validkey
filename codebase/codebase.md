# Keycheck Codebase

This folder contains documentation for the **validkey** CLI tool, which validates API keys for services such as OpenAI.

## Documentation index

- **[cli.md](cli.md)** – CLI interface: commands, options, usage, and output.
- **[validation.md](validation.md)** – How each service is validated (endpoints, HTTP status codes, error handling).
- **[services.md](services.md)** – List of supported services and how to add new ones.

When making changes, consult these docs to keep behavior consistent and avoid breaking existing usage.
